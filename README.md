Encrypted Notes App

Веб-приложение для безопасного хранения заметок с полным шифрованием на стороне сервера
Ключевые возможности

     Полное шифрование - AES-256-GCM алгоритм с аутентификацией данных

     Управление заметками - создание, просмотр, удаление зашифрованных записей

     Безопасность - пароли никогда не сохраняются, только зашифрованный контент

     Локальное хранение - SQLite база данных, не требует облачных сервисов

     Адаптивный интерфейс - работает на любом устройстве

 Технологический стек

    Backend: PHP 8.4+ с OpenSSL

    База данных: SQLite (или MySQL)

    Шифрование: AES-256-GCM (рекомендовано NIST)

    Интерфейс: Bootstrap 5, чистый CSS

    Архитектура: MVC-подобная с разделением логики

 Особенности безопасности

     Аутентифицированное шифрование (GCM тег)

     Уникальный IV для каждой заметки

     Нет хранения мастер-паролей

     Защита от подделки данных

     HTTPS готовность

 Быстрый старт

# 1. Клонировать репозиторий
```
git clone https://github.com/yourusername/encrypted-notes-app.git
```
# 2. Запустить PHP сервер
```
php -S localhost:8000
```

# 3. Открыть в браузере
```
# http://localhost:8000/init_database.php
# http://localhost:8000/
```
